# Logo

The WeatherBlend quatrefoil — four overlapping circles in the metric palette (lime, mint, blue, purple), the same shape as the data-driven `WeatherOrb`. Pair with the wordmark, whose "Blend" carries a mint→blue gradient.

```jsx
<Logo size={30} />
<Logo size={26} wordmark />
```

This is the real brand mark shipped with the product. Never redraw it — reuse this component or `assets/logo-quatrefoil.svg`.
