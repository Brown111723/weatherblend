# Button

The app's primary action control — solid accent-blue by default, with `secondary` (muted fill) and `ghost` (borderless) for lower emphasis.

```jsx
<Button onClick={save}>Use my location</Button>
<Button variant="secondary">Cancel</Button>
<Button variant="ghost" size="sm">Skip</Button>
```

Variants: `primary` · `secondary` · `ghost`. Sizes: `sm` · `md` · `lg`. Props: `fullWidth`, `disabled`, `icon`.
