Multi-model weather forecast · All data free, no API key required

What is it?

WeatherBlend fetches forecasts from multiple global weather models simultaneously and blends them. Research shows multi-model blends are consistently more accurate than any single model.

The cards view

The hero shows the selected day: temperature with feels-like, plus rain, wind and cloud figures. Below it, one chart draws the whole day — the temperature curve (solid where observed, dashed where forecast, wrapped in a soft ribbon that widens where the models disagree), blue rain bars rising from the floor, a faint cloud band across the top of the sky, night shading outside sunrise/sunset, and the day's high/low labelled right on the curve. A glowing dot rides the curve at the red "now" line. Hold and drag anywhere on the chart or the hero figures to scrub to any hour; flick to slide between days.

The metric cards and hourly strip always reconcile with the table — the same blended (or observed) value appears everywhere.

Reading the table

Each section heading row is the blended average. Open ☰ → Models & sources and enable Sources to see individual model rows below it, each tagged with its current accuracy weight. The ✓ Actual row shows Open-Meteo’s analysis for past hours. A Confidence row in each section shows the per-hour model agreement for that metric. Daytime columns are shaded lighter; a red column marks the current hour.

Accuracy & weighting

Models are blended by how accurate they've recently been at your location, scored per metric against Open-Meteo's analysis of hours already gone — the same ✓ Actual data the table shows. Skill is sharpened (inverse-squared error), shrunk toward equal until enough evidence accumulates, and clamped so no model is silenced or dominates.

Weight method: Recency — errors decay with age, so the last couple of days matter most. Per-day — scores each model per day over the last X days, each day counting equally. Blend — the mean of the two. The learning window sets how many past days are fetched and scored. ☰ → Forecast accuracy shows the full per-model scorecard and the blend's day-by-day record.

Confidence

Confidence is a plain-English read on how much the models agree. When they all predict much the same thing it's High; when they diverge it falls to Medium or Low. It's computed per metric, per hour and per day. It measures agreement, not certainty.

The forecast glyph

Instead of fixed weather icons, the now card draws one living mark built from the data itself: four circles — mint (temperature), lime (wind), blue (rain) and purple (cloud) — each growing and brightening with its value, with the strongest metric glowing. Every forecast gets its own visual fingerprint.

Location

Tap the location name to search any city (GPS first, saved fallback). Viewing a place in another timezone opens the cards and hourly strip on that location's current time.
