WeatherBlend

Multi-model weather forecast · All data free, no API key required

What is it?

WeatherBlend fetches forecasts from multiple global weather models simultaneously and averages them together. Research shows multi-model averages are consistently more accurate than any single model.

Timeframes

1hr — next 48 hours, one column per hour.
3hr — next 8 days, one column every 3 hours.
8hr — next 10 days, one column every 8 hours.
Daily — 10-day summary with max/min temp, total rain, max wind.

Reading the table

Each section heading row is the average. Open ⚙ Config and enable Sources to see individual model rows below it. Columns shaded slightly lighter are daytime hours. A vertical amber line marks the start of each new day.

Location

WeatherBlend tries browser GPS first. If denied, it falls back to IP-based location which may show your ISP city. Tap 📍 and type your city name for accurate location.

Config panel (⚙)

Display: toggle Temp, Rain, Wind sections and Sources rows on/off.
Models: enable or disable individual forecast sources. Disabled models are excluded from the average.

Data: Open-Meteo · GFS (NOAA/USA) · ECMWF IFS 0.25° · ICON (DWD/Germany) · GEM (Canada) · BOM ACCESS-G (AUS) · UKMO (Met Office) · CMA (China) · JMA (Japan) · No API key required. Auto-refreshes hourly.
